# Emmet-ის

Emmet საშუალებას გაძლევთ დაწეროთ HTML და CSS კოდის დიდი ბლოკები მარტივი აბრევიატურების აკრეფით და Tab ღილაკზე დაჭერით.

## 1. HTML აბრევიატურები

### საფუძვლები

| აბრევიატურა | შედეგი                          | აღწერა                                        |
| ----------- | ------------------------------- | --------------------------------------------- |
| !           | (სრული HTML5 შაბლონი)           | ქმნის სტანდარტულ HTML5 დოკუმენტის სტრუქტურას. |
| link        | <link rel="stylesheet" href=""> | CSS-ის სწრაფი დაკავშირება.                    |

### ID და კლასები (Classes)

როგორც CSS სელექტორები.

- `.` = კლასი (Class)
- `#` = ID

| აბრევიატურა   | შედეგი                               |
| ------------- | ------------------------------------ |
| .container    | `<div class="container"></div>`      |
| #header       | `<div id="header"></div>`            |
| p.text-red    | `<p class="text-red"></p>`           |
| h1#title.main | `<h1 id="title" class="main"></h1> ` |

### იერარქია (ჩალაგება)

| სიმბოლო | მნიშვნელობა          | მაგალითი    | შედეგი                          |
| ------- | -------------------- | ----------- | ------------------------------- |
| `>`     | **შვილი** (შიგნით)   | `div>ul>li` | `<div><ul><li></li></ul></div>` |
| `+`     | **და-ძმა** (გვერდით) | `h1+p`      | `<h1></h1><p></p>`              |

### გამრავლება და ნუმერაცია

იდეალურია სიებისა და Grid-ებისთვის.

- `*` - გამრავლება
- `$` - ნუმერაციის მთვლელი (1, 2, 3...)
- `$$` = ნუმერაცია ნულებით (01, 02, 03...)

#### აბრევიატურა & შედეგი

`ul>li*3`

```HTML
<ul>
   <li></li>
   <li></li>
   <li></li>
 </ul>
```

---

`ul>li.item$*3`

```HTML
<ul>
   <li class="item1"></li>
   <li class="item2"></li>
   <li class="item3"></li>
 </ul>
```

### ატრიბუტები და ტექსტი

დაამატეთ კონტენტი და ატრიბუტები მყისიერად.

- `{}` - ტექსტური კონტენტი
- `[]` - საკუთარი ატრიბუტები

| აბრევიატურა                      | შედეგი                            |
| -------------------------------- | --------------------------------- |
| `a{დააჭირე აქ}`                  | `<a href="">დააჭირე აქ</a>`       |
| `p>{Lorem Ipsum}`                | `<p>Lorem Ipsum</p>`              |
| `input[type="text"]`             | `<input type="text">`             |
| `img[src="logo.png" alt="Logo"]` | `<img src="logo.png" alt="Logo">` |

---

> ul.menu>li\*3>a[href="#"]{ბმული $}

```HTML
<ul class="menu">
    <li><a href="#">ბმული 1</a></li>
    <li><a href="#">ბმული 2</a></li>
    <li><a href="#">ბმული 3</a></li>
</ul>

```

## 2. Lorem Ipsum (სატესტო ტექსტი)

Emmet-ს შეუძლია დააგენერიროს შემთხვევითი ტექსტი თქვენი დიზაინის შესავსებად.

- **lorem** - აგენერირებს 30 სიტყვიან წინადადებას.
- **lorem10** - აგენერირებს 10 სიტყვას.
- **lorem\*3** - აგენერირებს 3 პარაგრაფს.

მაგალითი: `p*2>lorem10`

შედეგი:

```HTML
<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quae, distinctio.</p>
<p>Saepe, libero. Voluptates dolorum natus tempora sunt? Aspernatur, itaque!</p>
```

# ახალი HTML-ის თეგები

## 1. `<ul>` (Unordered List)

არადალაგებული სია გამოიყენება ჩამონათვალისთვის, სადაც თანმიმდევრობას არ აქვს მნიშვნელობა. ვიზუალურად ჩნდება შავი წერტილები (bullets).

## 2. `<ol>` (Ordered List)

დალაგებული (დანომრილი) სია გამოიყენება მაშინ, როდესაც თანმიმდევრობას აქვს მნიშვნელობა (მაგ: რეცეპტი, რეიტინგი). ვიზუალურად ჩნდება რიცხვები (1, 2, 3...).

## 3. `<li>` (List Item)

სიის ელემენტი ეს არის კონკრეტული ნივთი, რომელიც უნდა ჩაჯდეს `<ul>`-ის ან `<ol>`-ის შიგნით.

## მაგალითები:

> ბრაუზერში ნახეთ შედეგი.

**არადალაგებული სია (`<ul>`):**

```HTML
<ul>
  <li>პური</li>
  <li>ყველი</li>
  <li>კარაქი</li>
</ul>
```

**დალაგებული სია (`<ol>`):**

```HTML
<ol>
  <li>გაიღვიძე</li>
  <li>დაიბანე</li>
  <li>ისაუზმე</li>
</ol>
```

# CSS ერთეულები

## რატომ გვჭირდება სხვადასხვა ერთეულები?

- **ხელმისაწვდომობა (Accessibility):** მომხმარებლებს შეუძლიათ შეცვალონ ბრაუზერის ნაგულისხმევი შრიფტის ზომა (მაგალითად, მხედველობის პრობლემების გამო). Rem შესაბამისად შეიცვლება, პიქსელები კი არა.

- **რესპონსიულობა (Responsiveness):** ფიქსირებული 900px კონტეინერი "გატყდება" მობილურის 375px ეკრანზე. ფარდობითი ერთეულები, როგორიცაა % ან vw, ავტომატურად ადაპტირდება.

| ერთეული | ტიპი               | საუკეთესოა                               | მოერიდეთ                                       |
| ------- | ------------------ | ---------------------------------------- | ---------------------------------------------- |
| px      | აბსოლუტური         | ჩარჩოები, ჩრდილები, წვრილი ხაზები.       | შრიფტის ზომები, კონტეინერის სიგანეები.         |
| rem     | ფარდობითი (Root)   | შრიფტები, მარჟები (Margins), Padding.    | კომპონენტის სპეციფიკური მასშტაბირება.          |
| em      | ფარდობითი (Parent) | ღილაკები, კომპონენტის Padding.           | ჩადგმული სიები (სწრაფად იზრდება ზომაში).       |
| %       | ფარდობითი (Parent) | Grid-ის განლაგება, სურათები.             | შრიფტის ზომები.                                |
| vw/vh   | ვიუპორტი           | Hero სექციები, ფონი მთელ ეკრანზე.        | ტექსტი (შეიძლება გახდეს ზედმეტად პატარა/დიდი). |
| ch      | სიმბოლო            | ტექსტის სიგანის კითხვისთვის ოპტიმიზაცია. | განლაგების სტრუქტურა.                          |

# რესპონსიული CSS

## Media Query-ები

Media query-ები საშუალებას გაძლევთ გამოიყენოთ CSS სტილები მხოლოდ მაშინ, თუ გარკვეული პირობები სრულდება.

```CSS
@media media-type and (condition) {
    /* CSS წესები იწერება აქ */
}
```

### მედიის ტიპები (Media Types)

- `all` (ნაგულისხმევი): გამოიყენება ყველა ტიპის მოწყობილობისთვის.

- `screen`: გამოიყენება კომპიუტერის ეკრანებისთვის, პლანშეტებისთვის, სმარტფონებისთვის.

- `print`: გამოიყენება პრინტერებისთვის (მაგ: ნავიგაციის დამალვა ბეჭდვის დროს).

### პირობები (Features)

- `width`, `min-width`, `max-width`: ვიუპორტის სიგანე.

- `orientation`: portrait (ვერტიკალური) ან landscape (ჰორიზონტალური).

> ვრცელდება მხოლოდ იმ ეკრანებზე, რომლებიც მინიმუმ 600px სიგანისაა ან არის ჰორიზონტალურ (landscape) რეჟიმში

```CSS
@media screen and (min-width: 600px), (orientation: landscape) {
    body {
        background-color: lightblue;
    }
}
```

```CSS
.container {
    width: 100%;
    padding: 10px;
}

/* პლანშეტის სტილები (სირთულის დამატება) */
@media (min-width: 768px) {
    .container {
        width: 80%;
        margin: 0 auto;
    }
}

/* დესკტოპის სტილები */
@media (min-width: 1024px) {
    .container {
        width: 60prem;
    }
}
```

## გავრცელებული Breakpoint-ები

| მოწყობილობის კატეგორია     | ზომის დიაპაზონი | ლოგიკა (Mobile First)             |
| -------------------------- | --------------- | --------------------------------- |
| ექსტრა პატარა (ტელეფონები) | < 576px         | ნაგულისხმევი CSS (Query-ს გარეშე) |
| პატარა (დიდი ტელეფონები)   | >= 576px        | `@media (min-width: 576px)`       |
| საშუალო (პლანშეტები)       | >= 768px        | `@media (min-width: 768px)`       |
| დიდი (ლეპტოპები)           | >= 992px        | `@media (min-width: 992px)`       |
| ექსტრა დიდი (დესკტოპები)   | >= 1200px       | `@media (min-width: 1200px)`      |

> რესპონსიული სურათების წესიყოველთვის დარწმუნდით, რომ სურათები არ სცდება თავიანთ კონტეინერებს

```CSS
.img {
    max-width: 100%;
    height: auto;
    display: block;
}
```
